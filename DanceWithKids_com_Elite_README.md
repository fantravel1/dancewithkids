# 4) The 5 Pillars of DanceWithKids.com

## Pillar 1 — The Easy-to-Hard Tutorial Ladder (Crown Jewel)

Every dance routine exists in three tiers:

### Level 1 — Easy
- ages 3–6 (or beginners)
- simple repetition
- big movements
- rhythm basics
- follow-along mode
- 1–2 minute duration

### Level 2 — Moderate
- ages 6–10
- coordination combos
- rhythm changes
- directional transitions
- 3–5 minute duration

### Level 3 — Hard
- ages 9+
- combo stacking
- tempo variation
- light freestyle components
- performance-ready sequences
- 4–6 minute duration

Same song.
Same theme.
Different complexity.

This allows families with mixed ages to dance together.
